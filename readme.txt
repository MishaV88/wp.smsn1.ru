Интеграция СМС уведомлений, верификация, регистрация и авторизация по номеру телефона с использованием одноразового кода из СМС. СМС уведомления в том числе для WooCommerce. Пока только для сотовых операторов Российской Федерации.

== Description ==
= Основные возможности плагина для WordPress =
[Скорость и Безопасность]
Одноразовые пароли (OTP — one-time password) через СМС позволяют быстро, безопасно и удобно авторизовываться и/или регистрироваться на сайте.
Пользователям не нужно придумывать и запоминать пароль. Одноразовый пароль можно использовать только один раз, а срок его жизни составляет всего 5 минут.
[Кастомизация форм авторизации и регистрации]
Предусмотрена возможность копировать шаблоны форм «Войти», «Зарегистрироваться» и т. п. в тему сайта.
Это даёт возможность оформить формы под Ваш дизайн и не только, при этом не беспокоясь, что после обновления плагина они будут перезаписаны. Плагин разработан на основе чистого и поддерживаемого кода, который соответствует стандартам WordPress и WooCommerce. Это делает его удобным для расширения и модификации.
[Разместите формы входа и регистрации в удобном для Ваших пользователей месте]
Используя шорткоды, Вы можете разместить формы входа и регистрации в любом месте Вашего сайта*. Используя в том числе pop-up(всплывающие) окна.
*Не рекомендуется размещать форму входа и регистрации на одной странице.
[Защитите свой интернет-магазин]
На этапе оформления заказа в интернет-магазине(на плагине WooCommerce) активируйте функцию проверки номера телефона для не авторизованных пользователей. Это позволит предотвратить отправку поддельных заказов и обеспечит обратную связь в режиме реального времени. В случае ошибки в номере телефона клиент сможет оперативно её исправить.
[Настройте так как Вам нужно]
Плагин изначально сконфигурирован наилучшим образом. Однако при необходимости можно настроить параметры, которые позволят изменять продолжительность сессии авторизованного пользователя. Также можно настроить перенаправления после входа и выхода из личного кабинета, в том числе для страницы my-account от WooCommerce.
[Используйте функцию отправки SMS где хотите]
После установки плагина и ввода учетных данных Вы можете вызвать функцию отправки СМС в любом месте Вашего кода с помощью публичной функции:
wp_smsn1_ru_do_send_function('номер телефона', 'Мой текст');
Так же доступны: 
wp_smsn1_ru_get_balance() - возвращает баланс аккаунта(при условии что введены login и API KEY в настройках плагина).
wp_smsn1_ru_get_senders() - возвращает доступные имена отправителей для аккаунта(при условии что введены login и API KEY в настройках плагина).
[Бесшовная интеграция]
После активации плагина, создается две страницы с формами авторизации и регистрации на сайте. Вам достаточно опубликовать их и установить ссылки. Для проверки номера телефона на странице с классической формой оформления заказа Вам достаточно включить в настройках соответствующую функцию*.
*В актуальной версии плагина пока не реализована возможность создания блочной страницы оформления заказа. В дальнейшем эта функция будет добавлена.
[100 СМС после регистрации]
Плагин предоставляется бесплатно, оплата взимается только за отправленные СМС. Для тестирования и оценки плагина мы предоставляем 100 СМС бесплатно. Чтобы воспользоваться ими, после регистрации на основном сервисе smsn1.ru: direct.docs-group.ru, сообщите в службу поддержки свой логин.

== Установка и настройка ==
1. Установите плагин на свой сайт. Актуальную версию плагина вы всегда можете найти на GitHub: https://github.com/MishaV88/wp.smsn1.ru .

2. Зарегистрируйтесь в сервисе отправки СМС www.smsn1.ru. Чтобы получить доступ к личному кабинету, перейдите по ссылке https://direct.docs-group.ru/. Скопируйте Логин и API ключ из раздела API личного кабинета.

3. В настройках плагина wp.smsn1.ru на своём сайте введите Логин и API ключ.

4. Проверьте оставшиеся настройки плагина. По умолчанию они настроены оптимально. 
Важно! Если вы не используете редиректы со стандартных страниц авторизации wp-login.php и/или /my-account/ (стандартная форма авторизации для WooCommerce, редирект срабатывает, если пользователь не авторизован), рекомендуется установить плагин типа Loginizer (https://wordpress.org/plugins/loginizer/), чтобы предотвратить атаки методом подбора пароля.

5. Замените ссылки входа в личный кабинет на стандартную страницу авторизации плагина "Вход на сайт"(login-by-wp-smsn1-ru).

6. Если у вас возникнут вопросы, обратитесь в службу поддержки: https://wp.smsn1.ru/контакты-службы-поддержки/.
